## 下载

### Docker

```
docker pull apachecn0/apachecn-ds-zh
docker run -tid -p <port>:80 apachecn0/apachecn-ds-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install apachecn-ds-zh
apachecn-ds-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-ds-zh
apachecn-ds-zh <port>
# 访问 http://localhost:{port} 查看文档
```
